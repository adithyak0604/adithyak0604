## 🚀 Featured Projects

### 🌾 [AgriSense — Smart Farm Intelligence Dashboard](https://github.com/adithyak0604/AgriSense)
> `Python` `Flask` `TensorFlow` `PyTorch` `scikit-learn` `SQLite` `ESP32`

An intelligent agricultural platform that brings together **crop disease detection**, **real-time soil monitoring**, **weather insights**, and **ML-based crop recommendations** to help farmers make data-driven decisions.

- 🦠 Disease detection for Banana, Coffee, Corn, Mango & Paddy using ResNet, EfficientNet, DenseNet & CLIP ViT
- 🌱 Live soil sensor data via ESP32 (USB serial & Wi-Fi modes)
- 🌤️ OpenWeatherMap integration with disease risk scoring
- 📊 SQLite database with full historical analytics

---

### 😄 [Navarasa Detection — Facial Emotion Recognition](https://github.com/adithyak0604/Navarasa-Detection)
> `Python` `TensorFlow` `OpenCV` `PyTorch` `Hugging Face`

A real-time facial emotion recognition system that detects the **9 Navarasa** (classical Indian emotions) from live webcam or IP camera streams using deep learning.

- 🎭 Detects all 9 Navarasa emotions: Adbhuta, Bhayanaka, Hasya, Karuna, Raudra, Shanta, Shringara, Veera & Bibhatsya
- 📷 Supports webcam and IP camera input
- 🤖 Pre-trained Keras model hosted on Hugging Face
- ⚡ 3-second emotion lock for stable predictions
